F-Droid First & Open Source

Atrium was built to fill the niche left by LunaSea. It ensures reproducible builds with no proprietary blobs and no telemetry. Licensed under GPL-3.0-or-later, it’s a modern Flutter codebase that fully respects your privacy.

Unmatched Depth

Atrium isn't just a dashboard. It's a fully-featured controller for every service.

The *arr Stack
  • Sonarr & Radarr: Full settings editor, calendar, advanced sorting, and filtering.
  • Prowlarr: Add indexers, manual search & grab, history, and system status.
  • Bazarr: Manage wanted subtitles, manual search, and provider details.
Media Servers
  • Plex: Explore libraries, genres, now-playing controller, and music.
  • Jellyfin & Emby: Remote control capabilities, live sessions, libraries, and music.
  • Tautulli: 30-day stats, detailed history, and active stream termination.
Downloads & Utilities
  • qBittorrent: Real-time torrent list, add/manage, per-file priority, and tracker details.
  • Seerr: Discover new media, search, and manage requests (approve/decline/retry).
  • SABnzbd: Powerful queue control.
  • Glances: Live CPU, memory, network, and disk monitoring gauges.

Designed for Power Users

A meticulously crafted controller built for massive media stacks.

dns
Multi-Instance

Run two Sonarrs, three qBits, whatever. Every instance is a first-class profile entry.

route
Dual-URL Routing

Every instance has a LAN URL and a WAN URL. Probes LAN first, falls back to WAN automatically.

bolt
Unified Activity Feed

One tab aggregates active streams and live transfers (qBit downloads, SAB slots, *arr queues) across all instances.

calendar_month
Global Calendar

A beautiful month grid aggregating upcoming Sonarr airings and Radarr releases with quick status dots.

security
Reverse-Proxy Ready

Global and per-instance custom HTTP headers ensure seamless compatibility with Authelia and Cloudflare Access.

fingerprint
Secure & Native

API keys live in Android Keystore with optional biometric unlock. Wake-on-LAN packets sent with pure Dart UDP.
