# atrium (app)

The Flutter application package - this is what builds into the Atrium
APK. All feature code lives in the workspace packages
(`../packages/core_*`) and service modules (`../services/service_*`);
this package hosts the shell (router, dashboard, activity, calendar,
settings) and wires the modules together.

See the [repository README](../README.md) for project overview and build
instructions, and [CONTRIBUTING.md](../CONTRIBUTING.md) for the
contribution guide.
